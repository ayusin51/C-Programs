HTML stands for Hypertext Markup Language. It is used to design webpages. Tags are used to design these webpages.
Tags are mainly of 2 types :

  • Empty Tags or Unpaired tags : These tags do not require closing.
  • Paired tags : These tags require both opening and closing which is done using open and close tag. Closing is done by using "/".
Some tags have further properties. These properties are called attributes.

Horizontal Row Tag

It is used to give specifications to a horizontal row. It is an empty tag. It has the following attributes :
  • color : This attribute is used to give color to the text in that row.
  • size : This attribute is used to define the size or the height of the text in that row. It is in pixels.
  • width : This attribute is used to give widhth to that horizontal row. It is also in pixels.
  • align : This attribute is used to align the text in that row.

Paragraph Tag

This tag is used to create paragraphs. It needs to closed.

Heading Tag

This tag is used to give headings. It needs to be closed. On the basis of size of the heading, this tag is of 6 types :
  • < h1 >
  • h2
  • h3
  • h4
  • h5
  • h6
h1 has the largest size whereas h6 has the smallest size.
