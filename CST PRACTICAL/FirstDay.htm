HTML (HyperText Markup Language!!)

HTML stands for HyperText Markup Language. It is used to design Web Pages.
Tags are used in HTML for designing the web pages.Tags are written on < > brackets.
There are two types of tags : Paired Tags and Singular Tags.
Paired tags are those tags which require both starting and ending.The ending tag is also known as closing tag. The closing is done by inserting a "/" in front of the tag. Example : HTML tag, Head tag ,Bold tag,break tag etc.
Singular tags are those which need not to be closed. It is also known as stand alone tag. Example : Break tag, horizontal row tag, etc.
